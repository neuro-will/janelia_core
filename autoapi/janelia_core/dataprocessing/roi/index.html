janelia_core.dataprocessing.roi

Module for working with and representing regions of interest (ROIS) in imaging data.

Module Contents

ROI

Initializes an ROI object.
class janelia_core.dataprocessing.roi.ROI(voxel_inds: list, weights: numpy.ndarray)[source]

Initializes an ROI object.

Args:

voxel_inds: This is a tuple of length n_dims. Each entry contains either (1) the indices of each voxel for that dimension or (2) a slice denoting the sides of a hyper rectangle which defines the ROI. In both cases, dimensions are listed in the same order as the image the ROI is for.

weights: A numpy array of weights for each voxel. If all voxels have the same weight, this can be a scalar.

slice_roi(self, plane_idx: int, dim: int = 0, retain_dim=True)[source]

Returns a slice of an ROI.

Args:

plane_idx: The index of the plane to slice

dim: The dimension which defines the plane.

retain_dim: If true, the voxel_inds of the returned roi will be the same length as the original roi. If false, the entry in voxel_inds for the dimension that was sliced along will be removed.

Returns:
